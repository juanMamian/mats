Las expresiones algebraicas son expresiones numéricas especiales.

Una expresión numérica que incluye incógnitas, como las siguientes:

  • \(a \times 2\)
  • \(b \over 4\)
  • \((c+5) \times 3 \)
  • \({3 \times a} + {4 \times a} \)
  • \({b \over 2} + 3 \times c\)

Es llamada Expresión algebraica.

En las expresiones algebraicas se usa algunas reglas de notación especial para multiplicaciones:

  • Para escribir una multiplicación entre dos números explicitamente escritos, se usa un punto a media altura. Así:
    • \(3 \cdot 4\)
    • \(6 \cdot 2\)
  • Para escribir una multiplicación entre un número cualquiera y una expresión señalada con un paréntesis, no se usa símbolo. Así:
    • \(12 (4+2) \)
    • \(6 (t+6)\)
    • \(13 ({u \over 2 })\)
  • Para escribir una multiplicación entre un número cualquiera y una incógnita no se usa símbolo. Así:
    • \(6 a\)
    • \(8 h\)
    • \(2 x\)
